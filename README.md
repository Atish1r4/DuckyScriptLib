# 🦆 DuckyQuack: macOS Automation & Research Scripts

![License](https://img.shields.io/badge/license-MIT-green)

Набор скриптов на языке **Ducky Script** для быстрого аудита, сбора данных и автоматизации рутинных задач с использованием BadUSB устройств (Flipper Zero, Hak5 Rubber Ducky и др.).

## 🛠 Установка на Flipper Zero

1. Подключите SD-карту Флиппера к компьютеру или используйте **qFlipper**.
2. Перейдите в директорию `SD Card > badusb`.
3. Загрузите файлы `.txt` из этого репозитория в указанную папку.
4. На Флиппере откройте меню **Bad USB**, выберите нужный скрипт и нажмите **Run**.

## 💡 Важные нюансы (Troubleshooting)

### ⌨️ Раскладка клавиатуры
Ducky Script имитирует нажатия клавиш. Если на целевом Mac установлена **русская раскладка** как основная, команды превратятся в абракадабру. 
* *Решение:* Перед запуском убедитесь, что активен английский язык (US Layout).

### 🛡 Защита macOS (TCC)
Начиная с macOS Mojave, система блокирует доступ к папкам пользователя (`~/Library`). 
* В скриптах предусмотрена пауза (`DELAY 5000`), чтобы вы успели вручную нажать кнопку **"Разрешить"** в появившемся системном окне.

### ⏱ Тайминги
Если компьютер работает медленно (старые модели), увеличьте значения `DELAY` в скриптах, чтобы Spotlight и Терминал успевали прогружаться.
